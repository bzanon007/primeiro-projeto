# Meu Primeiro Projeto - Visualg

Exercícios de lógica de programação feitos no Visualg/Portugol.  
Objetivo: treinar algoritmos e lógica de programação.

## Exercícios
- CALCULO DE IMC.ALG:
  Algoritmo que solicita peso (em kg) e altura (em metros), calcula o IMC e informa a categoria do usuário de acordo com a tabela padrão de IMC.

- É PRIMO OU NÃO.ALG:
  Algoritmo que recebe um número e verifica se ele é primo, testando suas divisões e analisando se ele é divisível apenas por 1 e por ele mesmo.

- MELHOR ALUNO.ALG:
  Algoritmo que pergunta a quantidade de alunos, lê o nome e a nota de cada um e identifica o aluno com a maior nota ao final da execução.
  
## Como usar

Abra os arquivos `.ALG` no Visualg e execute.
