# Meu Primeiro Projeto - Visualg

Exercícios de lógica de programação feitos no Visualg/Portugol.  
Objetivo: treinar algoritmos e lógica de programação.

## Exercícios
- CALCULO DE IMC.ALG: Este algoritmo, pergunta seu peso em Kg, consulta tua altura em Metros, assim ele usa o calculo de IMC e define sua categoria baseado em uma tabela do IMC, apartir da "pontuação" que deu seu calculo de Kg e Altura, assim define em qual categoria você está.

- É PRIMO OU NÃO.ALG: Ele questiona um número, e apartir do 1, ele soma +1 até chegar no número selecionado, ele pega esse número e divide ele por todos os números anteriores ao número selecionado, para um número se primo, ele tem que ser divisivel por 1 e ele mesmo, exemplo do 3, quando o algoritmo for execultado, ele irá dar 1, 2 e 3; ele calculará se o 3, número selecionado é divisivel por 1, 2 e 3, como ele é divisivel por ele mesmo e 1, ele é primo, por que tem um comando que verifica se o resto da divisão é 0, se for 0 ele salva em uma variável como +1, caso essa informação passar de 2, ele define como não primo, caso não for, ele define como primo, pois deu 2 divisões desse número, o que define primo.

- MELHOR ALUNO.ALG: pergunta quantos alunos existem, assim define a quantia de repetições necessárias, pergunta nome e nota, assim a informação é guardada e sempre que há uma nota melhor, essa nota é jogada em "melhorNota", juntamente com "melhorAluno", assim quando a maior nota é definida o nome vai junto, já que foi guardado a informação juntamente com a nota, assim define o melhor aluno pela nota.

## Como usar

Abra os arquivos `.ALG` no Visualg e execute.
